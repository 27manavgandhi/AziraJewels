# Font licenses

Both typefaces bundled in `src/fonts/` are Google Fonts, licensed under
the SIL Open Font License 1.1. Full license text for each is included in
this folder.

- **Bodoni Moda** — `BodoniModa-OFL.txt`
  https://fonts.google.com/specimen/Bodoni+Moda
- **Jost** — `Jost-OFL.txt`
  https://fonts.google.com/specimen/Jost

Both are self-hosted (downloaded once, bundled with the project) rather
than loaded from Google's font CDN at runtime — no attribution notice is
required in the UI for OFL fonts, but the license text must travel with
any redistribution of the font files themselves, which is why this folder
exists.
